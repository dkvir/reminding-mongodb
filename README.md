## Setup

```bash
node --version
# v18.17.0

npm --version
# 9.6.7
```

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install

# nodemon
npm install nodemon -g
```

## Development Server

Start the development server on `http://localhost:4000`

```bash
nodemon app
```
