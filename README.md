## Setup

```bash
node --version
# v20.10.0

npm --version
# 10.2.3
```

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install

# nodemon
npm install nodemon -g
```
